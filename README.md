# Sales Analysis: Car Manufacturing
This repository contains a detailed analysis of sales data for a car manufacturer. The objective of this project is to understand customer behavior, identify regional performance gaps, and optimize sales strategies through RFM (Recency, Frequency, Monetary) segmentation.

# Summary of Indicators
The analysis highlights clear cyclical seasonality and distinct customer segments. The insights are divided into the following main points:

# Seasonality and Trends
Cyclical Demand: There is a clear cyclical pattern in sales, with significant peaks in November, driven by end-of-year demand.

Regional Performance: Sales vary significantly by city, indicating a need to review performance in underperforming markets.

# Customer Segmentation (RFM)
Recency and Frequency: The client base consists of 42% recent customers and 58% old customers, with a slightly higher percentage (51%) of low-frequency buyers.

Monetary Value: A significant portion of the base (76%) falls into the "low monetary value" category, suggesting room for growth.

# Strategic Recommendations
VIP Program: Implementation of a VIP plan for loyal clients to increase retention.

Post-Sales: Improvement in post-sales service specifically targeted at recent clients to foster long-term engagement.

Supply Chain: Adjustment of supply levels for low-frequency clients to stimulate purchase activity.

# Análise de Vendas: Fábrica de Automóveis
Este repositório contém uma análise detalhada dos dados de vendas de uma fábrica de automóveis. O objetivo deste projeto é compreender o comportamento dos clientes, identificar gargalos regionais de desempenho e otimizar as estratégias de vendas através da segmentação RFM (Recência, Frequência e Valor Monetário).

# Resumo dos Indicadores
A análise revela uma sazonalidade cíclica clara e segmentos de clientes distintos. A análise divide-se nos seguintes pontos principais:

# Sazonalidade e Tendências
Demanda Cíclica: Existe um padrão cíclico evidente nas vendas, com picos significativos em Novembro, impulsionados pela demanda de fim de ano.

Desempenho Regional: As vendas variam significativamente por cidade, indicando a necessidade de revisar o desempenho em mercados com menor volume.

# Segmentação de Clientes (RFM)
Recência e Frequência: A base é composta por 42% de clientes recentes e 58% de clientes antigos, com uma predominância (51%) de compradores de baixa frequência.

Valor Monetário: Uma grande parte da base (76%) encontra-se na categoria de "baixo valor monetário", sugerindo oportunidade de crescimento.

# Recomendações Estratégicas
Plano VIP: Implementação de um plano VIP para clientes fidelizados visando aumentar a retenção.

Pós-Venda: Melhoria no atendimento pós-venda voltada especificamente para clientes recentes, a fim de incentivar o engajamento a longo prazo.

Abastecimento: Ajuste nos níveis de abastecimento para clientes de baixa frequência, visando estimular a atividade de compra.
